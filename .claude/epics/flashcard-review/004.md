# Task: 更新首頁顯示今日待複習數量（index.html）

## Description

在現有 `public/index.html` 的首頁加入「今日待複習」Widget：

1. 頁面載入時呼叫 `GET /api/words?due=true` 取得待複習數量
2. 若 > 0：顯示「今日待複習 N 張」+ 連結到 `/flashcard.html`
3. 若 = 0：顯示「今天的複習已完成！🎉」

Widget 用 inline `<script>` 放在 `index.html` 尾端，不新增額外 JS 檔案。

## Acceptance Criteria

- [ ] 首頁顯示今日待複習單字數量
- [ ] 數量 > 0 時有可點擊連結，導向 `/flashcard.html`
- [ ] 數量 = 0 時顯示「今天的複習已完成！🎉」
- [ ] API 失敗時靜默隱藏 Widget（不中斷首頁其他功能）
- [ ] Widget 樣式與首頁整體風格一致（法語藍白主題）

## Technical Details

**新增至 `public/index.html` 的 HTML（放在適當位置，例如 `<main>` 頂部或導航區）：**

```html
<!-- 今日閃卡複習 Widget -->
<section id="flashcard-widget" style="display:none">
  <a href="/flashcard.html" id="flashcard-link" style="display:none">
    今日待複習 <strong id="due-count">0</strong> 張 →
  </a>
  <p id="flashcard-done" style="display:none">今天的複習已完成！🎉</p>
</section>
```

**新增至 `public/index.html` 尾端的 `<script>`（CommonJS 不適用，這裡是純瀏覽器 script，不加 type="module"）：**

```javascript
(async function () {
  try {
    const res = await fetch("/api/words?due=true");
    if (!res.ok) return;
    const data = await res.json();
    const count = data.count ?? (data.words ? data.words.length : 0);

    const widget = document.getElementById("flashcard-widget");
    widget.style.display = "";

    if (count > 0) {
      document.getElementById("due-count").textContent = count;
      document.getElementById("flashcard-link").style.display = "";
    } else {
      document.getElementById("flashcard-done").style.display = "";
    }
  } catch (_) {
    // API 失敗時靜默隱藏，不中斷首頁
  }
})();
```

**注意**：`GET /api/words?due=true` 回傳 `{ count: N, words: [...] }`，優先用 `count` 欄位；若無則用 `words.length` 作為 fallback。
