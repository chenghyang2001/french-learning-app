# Task: 建立閃卡頁面 HTML 結構與 CSS 3D 翻轉動畫

## Description

建立 `public/flashcard.html` 與 `public/css/flashcard.css`。頁面包含：閃卡容器（正面法文 / 背面中文）、翻面按鈕、「認識 / 不認識」按鈕、複習摘要區。翻轉動畫用純 CSS 3D transform，不需 JavaScript 計算。

本任務只做靜態 HTML/CSS，不含任何 fetch 邏輯（由 Task 003 負責）。

## Acceptance Criteria

- [ ] `public/flashcard.html` 存在，可從 `http://localhost:3000/flashcard.html` 訪問
- [ ] 卡片有 `.card-inner`（含 `.card-front` 法文面 + `.card-back` 中文面）
- [ ] 點擊「翻面」按鈕後，`.card-inner` 加上 `.flipped` class，觸發 CSS rotateY(180deg) 動畫
- [ ] 「認識」與「不認識」按鈕初始 `display: none`，翻面後才顯示
- [ ] 複習完畢後顯示 `#summary` 區塊（初始 hidden），含「認識 N 張 / 不認識 M 張」
- [ ] 樣式與現有 `public/css/` 風格一致（法語藍白主題）

**CSS 3D 翻轉（`public/css/flashcard.css`）：**

```css
/* 讓父容器提供 3D 視角 */
.card-container {
  perspective: 1000px;
  width: 340px;
  height: 200px;
  margin: 2rem auto;
}

/* 翻轉容器：transition 讓動畫流暢 */
.card-inner {
  position: relative;
  width: 100%;
  height: 100%;
  transition: transform 0.6s;
  transform-style: preserve-3d;
}

/* .flipped class 由 JS 切換，觸發翻轉 */
.card-inner.flipped {
  transform: rotateY(180deg);
}

/* 正面與背面共用：backface-visibility 隱藏背面 */
.card-front,
.card-back {
  position: absolute;
  inset: 0;
  backface-visibility: hidden;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  border-radius: 12px;
  font-size: 2rem;
  font-weight: bold;
}

.card-front { background: #e8f4fd; color: #1a5276; }
.card-back  { background: #1a5276; color: #fff; transform: rotateY(180deg); }

.hidden { display: none !important; }
```
